FAQ

The Virtual Observatory and Ecological Informatics System (VOEIS) is an end-to-end management solution for a wide variety of environmental data types. VOEIS provides a framework for data acquisition, quality assurance/control, visualization, and provenance tracking of temporally and spatially located data.
VOEIS was designed to meet the data control, curation, sharing and publication needs typical to research scientists working in the earth sciences. However, the flexible data model makes the software appealing to a very broad set of use cases and is in no way limited to use by researchers only.
VOEIS was developed with funding from the National Science Foundation’s (NSF) Experimental Program to Stimulate Competitive Research (EPSCoR), a capacity building program to enhance researchers’ and institutions’ ability to compete for federal research funds. This project is supported jointly by funding from EPSCoR programs in Montana and Kentucky. Currently, the Data Hub is hosted on a server at Montana State University. However, the software stack is freely available on GitHub.
If you work within the Montana University system, the Data Hub is free to use. Just contact the system administrator to set up an account and start managing your data!
VOEIS is currently used by numerous research groups within the Montana University system as a fully functional data management system. New features are being added all the time and the final version of the VOEIS software stack is expected in August of 2012.
If you would like to create and account and begin storing data in VOEIS, contact the Research Computing Group at Montana State University: http://rcg.montana.edu/contact/
