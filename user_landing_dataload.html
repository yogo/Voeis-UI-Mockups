Simultaneous Sample/Lab Analysis File Upload Wizard: Use this workflow to load data contained in a .csv file. File must include a column of Sample IDs, a column of timestamps (date/time that the sample was collected), and one or more columns of data values. Data representing unique variables must be oriented in columns. You do not need to create a record of sample IDs before you load laboratory analysis data using this workflow. Note: you will be unable to save sample-related metadata (sample_type, sample_medium, etc.) but you will be able to save metadata related to the particular lab analysis used to derive data values. Data contained in the uploaded file may only be from a single X,Y location.
